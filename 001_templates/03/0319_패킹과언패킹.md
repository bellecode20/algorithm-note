# 패킹(Packing) & 언패킹(Unpacking)

## 언패킹: `*`

리스트/튜플처럼 **묶여있는 값**을 함수 인자로 **풀어서** 넣을 때 쓴다.

```python
def add(a, b):
    return a + b

nums = [10, 20]
print(add(*nums))   # add(10, 20)
```

### 언패킹 다른 예시(좌표)

```python
alpha = (0, -1)

def rotate_point(r, c):
    return -c, r

print(rotate_point(*alpha))  # rotate_point(0, -1)
```

## 언패킹: `**` (dict)

딕셔너리를 함수에 넘길 때 **키워드 인자**로 풀어서 전달한다.

```python
def person(name, age):
    print(name, age)

info = {"name": "su", "age": 20}
person(**info)  # person(name="su", age=20)
```

## 변수 받을 때의 `*` = 남은 것 모으기(패킹)

왼쪽에서 `*변수`는 **남은 값들을 리스트로 모아 받는다**.

```python
a, *b = [1, 2, 3, 4]
print(a)  # 1
print(b)  # [2, 3, 4]
```

중간에도 가능.

```python
a, *b, c = [1, 2, 3, 4, 5]
print(a)  # 1
print(b)  # [2, 3, 4]
print(c)  # 5
```

문자열도 가능(문자 단위로 시퀀스 언패킹됨).

```python
a, *b = "hello"
print(a)  # h
print(b)  # ['e', 'l', 'l', 'o']
```

## 함수 정의에서의 패킹: `*args`, `**kwargs`

### `*args`

위치 인자를 여러 개 받아서 **튜플로 묶음**.

```python
def func(*args):
    print(args)

func(1, 2, 3)  # (1, 2, 3)
```

### `**kwargs`

키워드 인자를 여러 개 받아서 **딕셔너리로 묶음**.

```python
def func(**kwargs):
    print(kwargs)

func(name="su", age=20)  # {'name': 'su', 'age': 20}
```

## 함수 정의 vs 함수 호출 차이

- 함수 **정의**에서 `*args`, `**kwargs`: **모아서 받는 것** → 패킹  
- 함수 **호출**에서 `*lst`, `**d`: **풀어서 넣는 것** → 언패킹

```python
def func(*args):
    pass

nums = [1, 2, 3]
func(*nums)
```

## 자주 쓰는 예시

### 값 교환

```python
a, b = 10, 20
a, b = b, a
print(a, b)  # 20 10
```

### 함수에서 여러 값 반환

```python
def get_point():
    return 3, 5

x, y = get_point()
print(x, y)  # 3 5
```

### 리스트 합치기

```python
a = [1, 2]
b = [3, 4]
c = [*a, *b]
print(c)  # [1, 2, 3, 4]
```

### 딕셔너리 합치기

```python
a = {"x": 1}
b = {"y": 2}
c = {**a, **b}
print(c)  # {'x': 1, 'y': 2}
```

## 코테에서 자주 보는 형태(좌표)

```python
point = (2, 3)
r, c = point

def move(r, c):
    print(r, c)

move(*point)
```

## 자주 하는 실수

### 개수 안 맞음

```python
a, b = [1, 2, 3]  # ValueError
```

### `*` 없이 리스트/튜플을 그대로 넘김

```python
def add(a, b):
    return a + b

nums = [1, 2]
add(nums)    # TypeError (a, b에 안 맞음)
add(*nums)   # OK
```

### `**`는 dict(키워드 형태)만 가능

```python
def person(name, age):
    print(name, age)

info = ["su", 20]
person(**info)  # TypeError
```

## 한 문장 암기

**정의할 때의 `*`는 모으는 느낌, 호출할 때의 `*`는 푸는 느낌**
