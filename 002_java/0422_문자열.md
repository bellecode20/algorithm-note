char c = 'A';
String s = "A";

이 둘은 겉보기엔 비슷한데 타입이 다름.

1. char는 문자 1개만 가능
   char a = '가';
   char b = '1';
   char c = '+';

이건 가능.

근데

char x = 'ab'; // 오류

이건 안 돼.
char는 무조건 한 글자만 들어가야 해.

2. String은 문자열
   String s1 = "A";
   String s2 = "Hello";
   String s3 = "";

String은 글자 1개여도 되고 여러 개여도 돼.
심지어 빈 문자열 ""도 가능.

3. 그래서 비교할 때도 다름
   문자 비교
   char c = 'A';

if (c == 'A') {
System.out.println("같음");
}

char는 보통 ==로 비교해.

문자열 비교
String s = "A";

if (s.equals("A")) {
System.out.println("같음");
}

문자열은 == 말고 .equals() 를 써야 해.

이거 자바 초보가 제일 많이 틀리는 부분 중 하나야.
